# com.mersal.app

هذا المستودع يحتوي على هيكل مشروع Android (Kotlin) بسيط لتبدأ التطوير.

كيف تشغله محليًا

1. تأكد من تثبيت Android Studio وAndroid SDK.
2. استنساخ المستودع:

   git clone https://github.com/bolbolasha345-sketch/com.mersal.app.git
   cd com.mersal.app

3. افتح المشروع في Android Studio: File -> Open -> اختر مجلد المشروع.
4. انتظر Gradle ليعمل sync (قد يُطلب منك تثبيت أو تحديث Gradle أو الـ SDK).
5. شغّل التطبيق على محاكي أو جهاز فعلي عبر Run -> Run 'app'.

ملاحظات
- أضفت هيكل أساسي فقط (ملفات build.gradle، AndroidManifest، MainActivity، layout).
- إذا رغبت أضيف Gradle wrapper، أيقونات، موارد إضافية أو إعداد CI (GitHub Actions) أخبرني وسأفعّلها.
